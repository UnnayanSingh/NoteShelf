
# 📘 NoteShelf – Online Notes Sharing Platform

NoteShelf is a full-stack web application built using **Flask** and **MySQL** that allows users to explore and download subject-wise study notes in PDF format. It includes a secure admin dashboard for managing subjects and PDFs, along with user authentication and download tracking.

## 🚀 Features

### 👨‍🎓 User Features
- User registration and login
- Secure password hashing using Werkzeug
- Browse subjects without login
- View PDFs under each subject
- Login-protected PDF downloads

### 🔐 Admin Features
- Secure admin login (database-based, hashed passwords)
- Add, edit, and delete subjects
- Upload, update, and delete PDFs
- View download count for each PDF

### 📊 Analytics
- Tracks number of downloads per PDF

### 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML, Tailwind CSS
- Werkzeug
- Gunicorn

## ⚙️ Setup
1. Clone the repo
2. Install dependencies
3. Configure environment variables
4. Run the app using python app.py
